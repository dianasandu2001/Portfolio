BloodBloom

Genre: Horror, Exploration, Atmospheric, Open-World | Platforms: PC | Engine: Unity

Overview

BloodBloom is a open world horror game inspired by Resident Evil 2-4. The player plays a detective that is taking over the case of a missing detective in order to find him and catch those who kidnapped him. The game consits of a tutorial like scene where the game plot is set and a main area in which all of the action happens.
