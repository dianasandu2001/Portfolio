Endless Dreamer

Genre: Endless Runner | Platforms: PC | Engine: Unity

Overview

Endless Dreamer is an endless runner game inspired by Temple Run. The player collects gold, gems, power-ups and avoids obstacles and debuffs degined to hinder the player. The game consits of 5 unique characters with different powers and 4 different maps with unique aesthetics.

Development Details

This was my first ever finished product as well as the project for the Portfolio 1 course during my studies in second year at Metropolio, part of the Game Development major.

Since this was a solo project as well as my first ever project to complete, I used free assets from Maximo and the Unity Asset Store. The development took aproximately 3 months with feature refinements later on.

The Github Repository is public and contains: the unity project, the Game GDD & game pitch presentation, short description of the game, as well as a downloadable build zip file.

Gameplay Features

🌙Dream Worlds: Run through dynamically generated maps, each with unique themes like enchanted forests, floating cloudscapes, and mystical dungeons.
⚡Character Abilities & Progression: Unlock and upgrade characters, each with special skills to help navigate obstacles and maximize scores.
🧪Crafting System: Gather ingredients from different maps to craft potions that enhance abilities, remove debuffs, or grant extra lives.
🎵Dynamic Audio: Each map has its own soundtrack and sound effects that react to player actions.
🏆Score & XP System: Convert scores into experience points to level your characters and unlock rewards.
